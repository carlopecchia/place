Place

Extract information from PolarionALM™ turning it into easy manageable Ruby objects.

Place (Polarion Lace) collects informations from Polarion ALM â„¢ and turns it into easy manageable ruby objects, which are persisted by Redis key-value store.

"Information" includes: users, projects, work items, links, watches, comments, ...

source code gem rdoc

Install

  $ sudo gem install place

Example

Obtain a local working copy: 
  $ svn co http://your.polarion.server/repo local_repo
Access stuff with Ruby: 
  require 'place'
  include Place

  Place.setup('./local_repo')
  Place.logger_setup('log.txt')

  Place.gather!
  
  puts "Users: #{User.all.size}"
  puts "Workitems: #{WorkItem.all.size}"

  all_tasks = WorkItem.find_by_type(:task)

  demo = Project.find_by_name('Demo')
  demo_tasks = demo.workitems(:type => :taks)

  t = demo_tasks.last
  puts "Assignees for #{t.wid}: #{t.assignees.map{|u| u.name}.join(' ')}"
